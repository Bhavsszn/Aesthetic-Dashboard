# Financial Performance Dashboard (Power BI & Microsoft Fabric)

An interactive Business Intelligence dashboard built with **Microsoft Power BI** and **Microsoft Fabric** to analyze financial and operational performance through KPI reporting, executive dashboards, and interactive analytics.

## Overview

This project demonstrates the development of an end-to-end Business Intelligence solution using **Microsoft Fabric** and **Power BI**. The dashboard transforms raw business data into actionable insights through semantic modeling, DAX calculations, and interactive visualizations, enabling stakeholders to monitor performance and support data-driven decision-making.

---

## Features

- Executive KPI scorecards
- Interactive financial and operational dashboards
- Trend and performance analysis
- Dynamic filtering and slicers
- Drill-down and drill-through reporting
- Decomposition Tree analysis
- Interactive business intelligence visualizations
- Semantic data modeling using Microsoft Fabric

---

## Dashboard Pages

### Executive Dashboard
Provides a high-level overview of key business metrics through KPI scorecards and executive reporting.

### Performance Analysis
Interactive visualizations that allow users to explore business performance, identify trends, and compare key metrics over time.

### Decomposition Tree
Breaks down KPIs into contributing factors, enabling deeper analysis and root-cause investigation.
